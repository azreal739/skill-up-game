# 13 — Campaign Content Packs

## Purpose

This document defines the intended content packs.

## v1 Content Recommendation

Build Campaign 1 completely first.

Campaign 1 should prove:
- Content model works.
- Challenge engine works.
- Progression works.
- Help Centre works.
- UI feels good.

Then add later campaigns.
