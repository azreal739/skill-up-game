# Engineering Academy — Full Implementation Specification

Engineering Academy is a story-driven, browser-based learning game for an Angular/Nx front-end engineering team working with TypeScript, SCSS, Zod, Java APIs, AWS S3, CloudFront, Cloudflare, Git, CI/CD, testing and production support.

This repository is a specification pack intended for an AI coding agent such as Claude Code. It is intentionally written as a set of connected Markdown documents so the agent can use them as project context.

- Git
- CI/CD
- Unit testing and integration testing

## Build Philosophy

Build a polished product, not a demo. The game should feel like an engineering mission simulator with RPG progression and cinematic feedback.

Where this spec does not prescribe an implementation detail, the coding agent should make a clean, maintainable, idiomatic decision and document it in the codebase.
