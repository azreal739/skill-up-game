ENGINEERING ACADEMY — Mission Control
=====================================

A story-driven learning game. Everything runs on this computer — no
internet connection, accounts, or installation needed.

HOW TO START
------------

  Windows:  double-click  "Play - Windows.bat"
            (If Windows shows a blue "protected your PC" screen, click
             "More info" then "Run anyway" — it only does that because
             the file was downloaded.)

  Mac:      the FIRST time, right-click "Play - Mac.command" and choose
            "Open", then click "Open" in the warning dialog.
            After that you can just double-click it.

  Linux:    run  sh "Play - Linux.sh"  from a terminal (or double-click
            if your file manager allows executing scripts).

A black console window opens and your browser pops up with the game at
http://localhost:8377 — keep the console window open while playing
(minimising it is fine). Close it when you're done.

YOUR PROGRESS
-------------

Progress saves automatically in the browser you play in, on this
computer. So:

  * Always play in the same browser on the same computer.
  * Each family member should either use their own computer/browser
    profile, or you can share one save.
  * Clearing the browser's site data / "cookies and site data" for
    localhost will erase progress — careful with cleanup tools.

TROUBLESHOOTING
---------------

  * Browser opens but page says "can't connect": wait 2 seconds and
    refresh — the server may still be starting.
  * Nothing happens on double-click (Mac): right-click > Open (see above).
  * Want a different port? You can't — progress is tied to port 8377.
  * The game window closed but console still open: just browse to
    http://localhost:8377 again, no need to restart.
